# Recommendation Projects

Bu repository, çeşitli öneri sistemi (recommendation system) projelerini içermektedir.

## Proje Yapısı

```
RecommendationProjects/
├── Book_Recommendation/
│   └── BookRecommendation.ipynb
├── Job_Recommendation/
│   ├── JobRecommendation.ipynb
│   └── marketing_sample_for_naukri_com-jobs__20190701_20190830__30k_data.csv
└── Restaurant_Recommendation/
    ├── RestaurantRecommendation.ipynb
    └── TripAdvisor_RestauarantRecommendation.csv
```

## Projeler

### 📚 Book Recommendation
`Book_Recommendation/` klasöründe yer alan bu proje, kitap öneri sistemi geliştirmeye odaklanmaktadır.

### 💼 Job Recommendation
`Job_Recommendation/` klasöründe yer alan bu proje, iş ilanı öneri sistemi üzerine kuruludur. Veri seti olarak Naukri.com'dan alınan iş ilanları kullanılmaktadır.

### 🍽️ Restaurant Recommendation
`Restaurant_Recommendation/` klasöründe yer alan bu proje, TripAdvisor verilerini kullanarak restoran öneri sistemi geliştirmektedir.
